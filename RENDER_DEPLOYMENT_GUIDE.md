## 📋 OPTION 1: Deploy via Render Dashboard (Recommended)

### Step 1: Create Render Account
1. Go to: https://render.com
2. Click "Get Started" → Sign up with GitHub
3. Authorize Render to access your GitHub repos

### Step 2: Connect Your Repository
1. Click **"New +"** button → Select **"Blueprint"**
2. Connect your GitHub account (if not already connected)
3. Select repository: **`elite-wealth-capita`** or **`MY-SITE`**
4. Branch: **`main`**
5. Click **"Apply"**

### Step 3: Render Auto-Detects Configuration
Render will automatically:
- Read `render.yaml` blueprint
- Create PostgreSQL database (`elite-wealth-db`)
- Create web service (`elite-wealth-capita`)
- Set up Docker environment
- Configure auto-deploy

### Step 4: Set Environment Variables (IMPORTANT!)

After service is created, go to **Environment** tab and add:

**Required:**
```bash
# Already set by render.yaml, just verify:
DATABASE_URL=<auto-generated>
SECRET_KEY=<auto-generated>
DEBUG=False
ALLOWED_HOSTS=.onrender.com,elite-wealth-capita.onrender.com

# EMAIL (Set manually):
EMAIL_HOST_PASSWORD=<your-zoho-app-password>

# NEW SERVICES (Add these):
REDIS_URL=<get-from-upstash-or-render-redis>
STRIPE_SECRET_KEY=sk_live_...
STRIPE_PUBLISHABLE_KEY=pk_live_...
PAYPAL_CLIENT_ID=...
PAYPAL_SECRET=...
COINBASE_API_KEY=...
BYBIT_API_KEY=...
BYBIT_API_SECRET=...
CELERY_BROKER_URL=<redis-url>
CELERY_RESULT_BACKEND=<redis-url>
```

### Step 5: Deploy!
1. Click **"Manual Deploy"** → **"Deploy latest commit"**
2. Watch build logs in real-time
3. Wait 5-10 minutes for first deployment

### Step 6: Verify Deployment
1. Visit: `https://elite-wealth-capita.onrender.com`
2. Check homepage loads correctly
3. Test login/signup
4. Verify dashboard features

## 🌐 Custom Domain Setup (Optional)

### Add Your Domain:
1. Go to **Settings** → **Custom Domains**
2. Click **"Add Custom Domain"**
3. Enter: `elitewealthcapita.uk`
4. Add DNS records to your domain provider:
   ```
   Type: CNAME
   Name: www
   Value: elite-wealth-capita.onrender.com
   
   Type: A
   Name: @
   Value: <Render IP address shown>
   ```

5. Update `ALLOWED_HOSTS` in environment:
   ```
   ALLOWED_HOSTS=.onrender.com,elitewealthcapita.uk,.elitewealthcapita.uk
   ```

## ⚡ Performance Tips

1. **Enable Redis:** Essential for caching and Celery
   - Use Render Redis add-on OR Upstash free tier
   - Set `REDIS_URL` environment variable

2. **Use CDN:** For static files
   - Cloudflare, BunnyCDN, or Render CDN
   - Set `CDN_URL` and `CDN_MEDIA_URL`

3. **Optimize Images:**
   ```bash
   python build_tools/optimize_images.py
   ```

4. **Minify Assets:**
   ```bash
   python manage.py optimize_assets
   ```

## 🆘 Troubleshooting

### Build Fails:
- Check `requirements.txt` for missing/conflicting packages
- Verify `Dockerfile` Python version matches
- Check build logs for specific error

### Site Not Loading:
- Verify `ALLOWED_HOSTS` includes your domain
- Check environment variables are set
- Review application logs

### Database Connection Error:
- Verify `DATABASE_URL` is set correctly
- Check database service is running
- Ensure migrations have run

### Static Files Not Loading:
- Run `python manage.py collectstatic --noinput`
- Check `STATIC_ROOT` and `STATIC_URL` in settings
- Verify whitenoise is configured
